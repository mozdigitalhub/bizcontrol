# Glossário Canónico PT-PT (AO90)

Este glossário define os termos obrigatórios de interface e comunicação do BizControl.

## Substituições obrigatórias

- `cadastro`, `cadastrar` -> `registo`, `registar`
- `usuário`, `usuarios`, `usuario` -> `utilizador`, `utilizadores`, `utilizador`
- `senha` -> `palavra-passe`
- `faturamento` -> `faturação`
- `arquivo` -> `ficheiro`
- `deletar` -> `eliminar`
- `contato` -> `contacto`
- `endereco` -> `endereço`
- `registrar` -> `registar`

## Regras

- Aplicar estes termos em templates, emails, mensagens `messages.*`, labels de formulários e mensagens de validação de API.
- Não alterar chaves técnicas, slugs, nomes de permissões, URLs ou identificadores.
- Manter consistência de acentuação e capitalização em todo o texto visível ao utilizador.
